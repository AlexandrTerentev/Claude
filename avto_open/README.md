
# AvtoOpen

Кроссплатформенный лаунчер сборок с поддержкой автоматизации для Windows и Linux.

## Возможности

- Создание "сборок" - наборов приложений с настроенными действиями
- Параллельный запуск нескольких приложений
- Автоматизация через эмуляцию ввода (клики, ввод текста, горячие клавиши)
- Локальное хранение конфигурации

## Установка

```bash
# Клонирование репозитория
git clone <repo-url>
cd avto_open

# Установка зависимостей
pip install -r requirements.txt

# Для Linux может потребоваться:
# sudo apt install python3-tk python3-dev
```

## Запуск

```bash
python src/main.py
```

## Использование

1. **Создать сборку** - нажмите "Создать" и введите название
2. **Добавить приложения** - в редакторе сборки нажмите "Добавить" и выберите исполняемый файл
3. **Настроить действия** - дважды кликните на приложение для редактирования действий:
   - Задержка
   - Клик мыши (с помощником захвата координат)
   - Ввод текста
   - Горячие клавиши (например: `ctrl+shift+s`)
   - Прокрутка
4. **Запустить сборку** - выберите сборку и нажмите "Запустить"

## Пример

Создание сборки для стриминга:
1. Discord → задержка 3 сек → клик по чату
2. OBS → задержка 2 сек → горячая клавиша F9

```
avto_open/
├── src/
│   ├── main.py           # Точка входа
│   ├── core/             # Бизнес-логика
│   ├── ui/               # Интерфейс (PySide6)

```

## Зависимости

- Python 3.11+
- PySide6 (Qt GUI)
- PyAutoGUI (автоматизация)

## Конфигурация

Сборки сохраняются в:
- Linux: `~/.config/avto_open/assemblies.json`
- Windows: `%APPDATA%/avto_open/assemblies.json`
